# DiabetEase

**Huawei Developer Competition 2025 APAC**

A comprehensive Type 2 diabetes care companion that empowers patients with AI-driven personalization, health tracking, and educational resources.

## Overview

DiabetEase is a full-stack application designed to support individuals managing Type 2 diabetes through intelligent food recommendations, health monitoring, and evidence-based educational content. Built with Next.js and powered by Huawei Cloud services, the platform delivers a scalable, secure, and personalized diabetes management experience.

## Key Features

### FoodScanner
Upload a photo of food ingredients and receive instant AI-powered meal suggestions that adhere to the plate method specifically designed for Type 2 diabetes management. Each recommendation includes detailed cooking instructions and comprehensive nutritional values.

### Daily Glucose Tracker & Analytics
Log and monitor blood glucose levels with intuitive visualizations and trend analysis to help you understand patterns and make informed health decisions.

### Curated Medical News
Stay informed with daily curated diabetes and medical news tailored to Type 2 diabetes care and management.

### Diabetic Care Center Locator
Find nearby diabetic care centers using geolocation-based recommendations to connect with healthcare providers in your area.

## Technology Stack

### Frontend
- Next.js

### Backend & Cloud Infrastructure
- **Huawei ModelArts** - AI-powered food recognition and personalized recommendations

## Getting Started

### Prerequisites
- Node.js (v16 or higher)
- npm or yarn package manager

### Installation

1. Clone the repository
```bash
git clone https://github.com/yourusername/diabetease.git
cd diabetease
```

2. Install dependencies
```bash
npm install
```

3. Run the development server
```bash
npm run dev
```

4. Open your browser and navigate to `http://localhost:3000`
